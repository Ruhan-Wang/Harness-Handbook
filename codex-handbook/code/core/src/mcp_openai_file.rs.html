//! Bridges Apps SDK-style `openai/fileParams` metadata into Codex's MCP flow.//!//! Strategy://! - Inspect `_meta["openai/fileParams"]` to discover which tool arguments are//!   file inputs.//! - At tool execution time, read those files from the primary environment,//!   upload them to OpenAI file storage,//!   and rewrite only the declared arguments into the provided-file payload//!   shape expected by the downstream Apps tool.//!//!
