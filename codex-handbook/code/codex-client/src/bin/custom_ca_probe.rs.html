//! Helper binary for exercising shared custom CA environment handling in tests.//!//! This//! probe keeps the behavior under test while letting integration tests (`tests/ca_env.rs`) set//! env vars per-process, proving://!//! - env precedence is respected,//! - multi-cert PEM bundles load,//! - error messages guide users when CA files are invalid.//! - optional HTTPS probes can complete a request through the constructed client.//!//!
